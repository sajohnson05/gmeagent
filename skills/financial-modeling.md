# Financial Modeling Skill

## Capabilities
- Forecasting
- Scenario modeling
- Budgeting
- Cash flow analysis

## Standards
- Driver-based models only
- No hardcoded values
- Clear assumptions
- Scalable structure

## Outputs
- Budget models
- Forecast scenarios
- Medicare funding models
- Growth projections

## Best Practices
- Separate inputs, calculations, outputs
- Stress test assumptions
- Document logic clearly
