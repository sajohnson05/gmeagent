# Executive Mode Guide

## Purpose

Use this when operating at VP / DIO level.

Focus:
- strategy
- resource allocation
- risk
- system performance

---

## How to prompt

Always include:

- decision context
- constraints
- desired outcome

Example:

Act as the GME executive team.
Evaluate current GME structure.
Recommend:
- org changes
- KPI improvements
- risks

---

## Key Use Cases

### 1. Org Design

Prompt:
Evaluate current GME org structure and recommend improvements.

### 2. KPI Review

Prompt:
Assess KPI framework and identify gaps.

### 3. Strategic Planning

Prompt:
Create 3-year GME strategy.

### 4. Risk Management

Prompt:
Identify top operational and accreditation risks.

---

## What good looks like

- clear decisions
- tradeoffs explained
- risks identified
- actionable next steps
