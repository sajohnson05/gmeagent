# Medicare Funding Flow

## Objective
Optimize DGME and IME reimbursement.

## Steps
1. Track resident FTEs
2. Validate cost data
3. Model reimbursement
4. Identify optimization opportunities
5. Report to leadership

## Agents Involved
- Finance Agent (lead)
- Data Agent (support)
- Executive Agent (decision)

## Outputs
- Funding model
- Forecast scenarios
- Optimization strategies
