# Core Skill Matrix

## Purpose
Defines the core skills expected across the GME office and highlights where strength is most critical.

| Skill                  | Required Across Roles | Notes |
|-----------------------|-----------------------|-------|
| Communication         | High                  | Essential across all leadership and support roles |
| Presentation          | Medium-High           | Important for faculty development, executive reporting, and strategy |
| Data Literacy         | Medium-High           | Needed for dashboards, decision-making, and compliance awareness |
| Project Management    | High                  | Required for cross-functional coordination and sustained execution |
| Relationship Building | High                  | Important across resident, faculty, and executive-facing roles |
| Regulatory Knowledge  | High                  | Critical for accreditation, policies, and Medicare-related functions |
| Financial Acumen      | Medium                | Important for leadership, strategy, and finance-facing roles |

## Application
This matrix should inform:
- hiring profiles
- role design
- staff development
- agent expectations
- leadership capability planning

## Design Standard
The office should not rely on one person holding all critical knowledge. Capability should be distributed and reinforced across the structure.
